# 开篇：关于学习、分享与感谢

> 小注：博客头图是我养的小猫，奶糖中时候、小时候 🐱

本科初期，我对 CS、编程并没有太多热情，转专业也是同样的原因没有选择当时大热的计算机、电子信息等等。
在大一（其实是进入复旦的第二年） Python 课程反复的 debug 过程中，几乎消磨掉了所有对 coding 的耐心。

直到 2022—2023 年左右，AI 辅助编程工具的出现，真正改变了这一切。
它极大降低了编程的门槛，也在很大程度上点燃了我对算法与代码的兴趣。

我的学习笔记、博客内容，深受四位拥有开源精神的分享者影响：

一位是高我两届的复旦统计学长，一位是高我一届的本专业学姐。
两位都非常优秀，目前在大藤进修金工（自惭形秽。
他们为本专业的多门核心课程留下了高质量笔记——非参数统计、统计推断、统计计算、机器学习……
这些内容极大方便了后来同学的学习，而完成这一切背后的工作量，更令我由衷敬佩。（他们做这些分享时，还是前AI时代

另外两位来自互联网社区：
一位是薇薇姐（喂薇薇），同样是高我一届的学姐，完整记录并分享了 CS336 的全部作业代码与课程笔记，笔记质量之高令人咋舌，内容详尽、体系完整，让我受益匪浅。
还有一位是年轻的学妹，本科三年内累计分享了 1000+ 篇内容，从 LeetCode 算法、LLM Infra 到 Linux 操作系统，涉猎极广、更新极勤，真切意识到自己与他人的差距，也时刻提醒自己不能怠惰。
